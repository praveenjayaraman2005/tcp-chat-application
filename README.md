# TCP Socket Two-Way Chat Application

A simple two-way chat application built using Python TCP sockets, 
enabling real-time communication between a client and server over a local network.

## 🔧 Technologies Used
- Python
- TCP Sockets
- Local Wi-Fi / LAN

## 📌 Features
- Real-time two-way messaging between client and server
- Socket creation and connection handling
- Message exchange over local network

## 🚀 How to Run

### Step 1 — Run the Server
```python server.py```

### Step 2 — Run the Client (in a new terminal)
```python client.py```

### Step 3 — Start Chatting!
Type messages in either terminal and they will appear on the other side.

## 📋 Requirements
- Python 3.x
- Both devices must be on the same Wi-Fi/LAN network
